# Web101-Event-Website-Project 

A single-page event RSVP website built with HTML, CSS, and JavaScript. It allows visitors to view event information and submit an RSVP through the page interface.

## Live Site
https://johannadev738.github.io/Web101-Event-Website-Project/

## Repository
https://github.com/johannadev738/Web101-Event-Website-Project

## Features
- Event information display
- RSVP interaction form
- Light and dark mode support
- Responsive single-page layout

## How to Run
1. Clone or download the repository.
2. Open `index.html` in a web browser.

## Project Structure
- `index.html` - main webpage
- `styles.css` - styling
- `script.js` - RSVP behavior
- `darkmode.js` - theme switching
- `images/` - image assets
